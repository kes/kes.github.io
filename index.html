FRACTRAN: John Conway’s Programming Language Fun

2025-11-16[Karl Stump]
#euler  #lisp 
Some months back I learned of an esoteric programming languages created by mathematician John Conway. Conway also created the well known Game of Life. The language is called FRACTRAN (see here for a Conway lecture). Very simply a FRACTRAN program is an ordered list of positive fractions \[ f_1, f_2, … f_a \] and an initial integer, \( n \). A FRACTRAN program is run in the following manner: for the first fraction \( f_i \) in the list for which \( n * f_i \) is an integer, replace \( n \) by \( n * f_i \) repeat the first rule until no fraction in the list produces an integer when multiplied by \( n \), then halt.
